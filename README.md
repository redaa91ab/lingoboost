# 📚 LingoBoost – Projet d’admission OpenClassrooms

**LingoBoost** est un outil développé dans le cadre de la **procédure d'entrée à la formation "Développeur d'application Python" proposée par OpenClassrooms**.

L’objectif : créer un projet web complet en moins de 72 heures, démontrant la maîtrise des bases du développement HTML, CSS, JavaScript, des algorithmes, ainsi que de Git/GitHub.

---

## 🎯 Objectif du projet

LingoBoost est un outil destiné aux apprenants francophones en anglais. Il permet d’analyser un document PDF (film, série, livre, etc.) et d’extraire automatiquement les **mots trop complexes pour le niveau CECR choisi (A1 à C1)**.

L’apprenant peut ainsi apprendre le vocabulaire difficile avant de lire ou regarder son contenu, et ainsi mieux comprendre sans frustration.

Le projet fonctionne hors ligne, sans connexion Internet, grâce à une base de données locale.

---

## 🚀 Fonctionnalités

- Sélection du niveau d’anglais (A1 à C1)
- Upload d’un fichier PDF
- Détection des mots difficiles pour le niveau sélectionné
- Traduction française automatique (via base locale)
- Interface simple, rapide et 100 % offline

---

## 🛠️ Technologies utilisées

- HTML5 / CSS3
- JavaScript
- [pdf.js](https://mozilla.github.io/pdf.js/) pour l’analyse des fichiers PDF
- Base de données locale de vocabulaire (CSV triés par niveau CECR)

---

## ▶️ Comment tester le projet

1. Lance `index.html` via un **Live Server** (par exemple dans Visual Studio Code avec l’extension Live Server)
2. Choisissez votre niveau d’anglais
3. Importez un fichier PDF
4. Découvrez le vocabulaire à apprendre

---

## 📁 Contenu du dossier

- `index.html` – Page principale
- `script.js` – Logique de traitement des mots
- `style.css` – Mise en page simple
- `/data/` – Vocabulaire CECR (A1 à C1) avec traductions françaises
- `captures/` – Captures d’écran des 3 cours OpenClassrooms réussis

---

## ✅ Cours requis (et validés)

- ✔️ Apprenez à créer votre site web avec HTML5 et CSS3 (15 heures)  
- ✔️ Découvrez le fonctionnement des algorithmes (6 heures)
- ✔️ Gérez du code avec Git et GitHub (6 heures)

Des captures d’écran des attestations sont incluses dans le dossier `captures/`.
